FAQ

En este apartado es donde podríamos responderte algunas preguntas que tengas actualmente.


  • ¿Qué es este lugar?

      • Este lugar es un generador de sitios estáticos, almacenamos todos los documentos de la página web en Markdown para que el generador haga el trabajo de convertirlos a HTML y, seguidamente, nosotros nos encargamos de ponerlos "bonicos".


  • ¿En estos momentos se puede adquirir algún producto?

      • No, actualmente esta página es una prueba que se está llevando a cabo para ver todo el posible potencial desplegable usando Hugo como generador de sitios estáticos.


  • ¿Qué son los "tags"?

      • Los tags es una característica añadida a la página web en el cual según el tag que cliques (Basic, Standard o Premium), te seleccionará los packs que tengan esa calidad que hayas clicado.


  • ¿Por qué filtrar por "Basic", "Standard" o "Premium"?

      • ¡Esas son las calidades de los packs! Según la calidad, entran más o menos contenido:

      Basic -> 2 objetos

      Standard -> 3 objectos

      Premium -> 4 objetos


  • Si es una página de pruebas, ¿Puedo usarla para realizar mis pruebas también?

      • ¡Por supuesto! Todo lo que sea compartir es bienvenido, por lo tanto, si deseas usar alguna parte de esta plantilla y quieres implementarlo a la tuya, eres libre de hacerlo.
